Module Hierarchy

 

In this lab, you will add two new module classes called TransformerType1 and PipeRackType1 into the Module Type catalog hierarchy. To create the classification folders as shown below, you must edit the ModuleType Excel workbook, which contains three sheets that control the hierarchy: ClassNodeType, R-Hierarchy, and R-ClassNodeDescribes. When you add a module class to the catalog hierarchy, you must update these sheets, where necessary, in order to see the data in the Catalog task or Catalog browser dialog box.

 

CatalogRoot is the RootNode of the ReferenceData Catalog Browser. It cannot have a parent node. It is a named-object. Similarly RefDataModulesRoot is the root for the module  hierarchy which appears as a child of the CatalogRoot.  Electrical, Transfomers, Structure and PipeRacks are the nodes to facilitate the easy browsing. TransformerType1 and PipeRackType1 are the Module Classes.

Lab 4: Part Class Modification

 

Objective

After completing this lab, you will be able to:

 

·          Modify a Part Class Definition

·          Preset Parametric Values

 

Equipment Class Modification

A part class definition can be modified for different reasons.  Some examples may include addition of new custom attributes or to redefine parametric values per approved vendor drawings.

This lab will step through the process of adding a new attribute into an equipment properties interface. Also parametric dimensions will be preset into the interface.

 

 

1.      Open the Equipment.xls Excel workbook in your working folder

2.      Access the BoilerType1 worksheet

3.      At the end of “Definition” row enter: oa:InspectionPoints  (see BoilerType2 for reference)
Note: This will insert InspectionPoints in the occurrence attributes (oa) form

4.      At the end of “Head” row enter: InspectionPoints  (see BoilerType2 for reference)
Note: This defines a column under which default values may be entered

5.      Copy the complete row for equipment definition “BoilerType1_IMP-E” and insert it in the line below to create a new equipment item definition for this part class

6.      Change the Name attribute in the copied equipment line to “BoilerType1_Lab”

7.      Update the PartDescription value for this equipment to “New Boiler”

8.      For the same item, enter “800lbm” under DryWeight

9.      Enter “10ft” under VesselLength, and “4.25ft” under VesselDiameter

10.  Set all Support parameters to “1in” (can’t be 0)

11.  Set nozzles 1 and 2 NPD parameters to 4

12.  Set nozzles 3, 4, and 5 to blank by deleting their parameters cells contents

13.  Set InspectionPoints value to 20

14.  In column A row 8 of the worksheet enter an “m” to modify the part class to include the InspectionPoints parameter as shown in the illustration

15.  In column A in the row defining the new boiler enter an “a” to append this new item into the equipment catalog as shown in the illustration


16.  Save the changes and use the Bulkload Utility in A/M/D mode to load the new boiler and modify the part class

17.  Enter SP3D Equipment task and place new boiler “BoilerType1_Lab”
