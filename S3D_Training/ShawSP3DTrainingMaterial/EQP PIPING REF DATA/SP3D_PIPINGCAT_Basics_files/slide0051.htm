Implied Data Troubleshooting
•This error indicates that part placement will be successful, but the implied gasket(s) for the bolted end(s) are not found in the catalog
•File: <NamePrefix>_SpecificationData.xls
•Worksheet: GasketSelectionFilter, DefaultProjectOptions
•File: <NamePrefix>_CatalogData.xls
•Worksheet: GasketPartData
•Action:
–Verify that part’s end prep data (Press. Rating, End Prep, End Std) have a match in the inputs (yellow) section of the GasketSelectionFilter worksheet
–Make sure the gasket commodity code in GasketSelectionFilter has a match in GasketPartData worksheet
–Set GsktSlnForMismatchBltdEndsOpt option = 10 in Default-ProjectOptions to allow for mismatched end preps
