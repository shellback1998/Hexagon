Creating a Custom Piping Spec
•To create a custom piping specification:
–Use procedure highlighted in previous slides to locate a delivered spec and rules that roughly resemble the new spec
–In “SpecificationData” workbook, name spec and redefine values needed by the new spec
(For details see pg 22 of C:\Program Files\Common Files\Intergraph\SmartPlant\Help\PipingReferenceData.pdf)
–In “SpecificationData” workbook set the spec rules for the new piping spec
–In “SpecificationData” workbook copy delivered spec commodities into PipingCommodityFilter worksheet
