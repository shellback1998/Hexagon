LAB-1: Out of the Box Iso Extraction

 

1.       Enter SmartPlant 3D and switch to Drawings and Reports task

2.       Right click on plant SP3DTrain, select New... to create a Folder



3.       Rename the new folder to: ISOS

4.       Right click on ISOS, select New... to create a Drawing By Query Manager from the General tab and OK



5.       Rename the resulting Drawing By Query Manager folder to Unit01

6.       Right click Unit01 and select Setup

7.       Under Filter select More... and pick Training Filters>U01

8.       Under Packages Select More... and pick Iso Pipeline then OK


9.       Right click Unit01 and select Run Query

10.   Expand the tree structure just created under Unit01, right click Iso Pipeline and launch Create Drawings



11.   Right click iso 1001-P and select Update Now

12.   Once the green check appears, right click drawing 1001-P and select Edit to review contents. Exit when done reviewing the drawing.



13.   Right click drawing 1001-P and select View Extraction Data

14.   Review log, pcf and isosheet drawing files

15.   STOP here, return to the presentation slides, and continue to the next slide

 

back to top.. i

 

LAB-2: User Graphics and Text

 

1.       Enter SmartPlant 3D and switch to Drawings and Reports task

2.       Navigate to iso drawing 1001-P from Lab 1 and open in Edit mode



3.       Add revision clouds around the gate valves indicating a revision to the operators orientation.

a.       Make sure to match the drawing areas active font

b.      Make sure to place manual items in Default or User layer



4.       Save and exit the drawing

5.       Run Update now on this drawing

6.       Return to drawing 1001-P in edit mode and verify that manual graphics were preserved after iso update.

7.       STOP here, return to the presentation slides, and continue to the next slide

back to top.. i

 

LAB-3: Title Block

 

1.       Return to drawing 1001-Ps parent folder Unit01, right click on it and select Properties



2.       In the Title Area tab, select the value field for Company Name and enter SHAW GROUP



3.       Scroll to the bottom of the Title Area form and set the value of Title 1 to UNIT01 ISOMETRICS

4.       Select the Signature Area tab

5.       Set the initials and dates for Drawn/Designed/Checked/Approved-by fields

6.       Select OK to accept new values

7.       Multi-select drawings 1001-P and 1002-P, right click and select Update now


8.       Review the contents of the title blocks for both isos, they should resemble the following illustrations:




Note: In practice, the title block layout may look different per project requirements.

9.       For iso 1002-P only, override the checkers initials from your default to XYZ

a.       Right click on drawing 1002-P and select Properties

b.      Select the Signature Area tab

c.       Click on Checked By value field and enter XYZ

Notice the Behavior column set to Override

d.      Run Update now on iso 1002-P and review the results in the title block

STOP here, return to the presentation slides, and continue to the next slide

LAB-4: Isometrics by Piperun

 

  1. In the ISOS folder, create a new Drawing by Query Manager component and rename it Unit02-Piperun



  2. Right click Unit02-Piperun and select Setup
  3. In the select package dialog, select the Iso Piperun package (if not found, CAE will provide it)
  4. In the Filter dialog select more
  5. Expand the Training Filters folder, select the filter U02 and OK


  6. OK to accept the Setup form
  7. Right click Unit02-Piperun and select Run Query. A folder tree is created
  8. Navigate down to the Process folder just created, right click it and select Create Drawings
  9. Expand pipeline folder 2002-P, select the isos by piperun component with a right click and select Update Now


  10. Review the contents of the updated drawings. Notice theres only one piperun per drawing. This method of extraction can be useful for users to check their design against Isogen, as well as diagnosing problems where system or pipeline isos are prevented from extracting successfully due to a problem in one undetermined piperun.
  11. STOP here, return to the presentation slides, and continue to the next slide

back to top.. i

 

LAB-5: Stress Isometrics

 

  1. In the ISOS folder, create a new Drawing by Query Manager component and rename it Unit02-Stress
  2. Right click Unit02-Stress and select Setup


  3. Set the Filter to unit U02 and the Package to Iso Stress


  4. Right click Unit02-Stress and select Run Query. A folder tree is created
  5. Navigate the new folders to the Iso Stress component. Right click it and select Create Drawings
  6. Notice that the Pipeline name drives the stress isos naming


  7. Right click iso 2001-P and select Update Now
  8. Since stress isos dont create a graphic file, right click the updated iso and select View Extraction Data


  9. In the Extraction Data form select PCF to view the stress analysis input data


  10. To export the stress analysis input file (PCF), right click on iso 2001-P and select Save as


  11. Enter an output folder, i.e., C:\LABS, where the report should be exported to
  12. For Isometric Drawings under Component Type, click the Target File Type pull down, select PCF file and OK


  13. In the Windows Explorer navigate to C:\LABS and review the file 2001-P.pcf
    This file would be the input into CAESAR II for piping stress analysis.
  14. STOP here, return to the presentation slides, and continue to the next slide

back to top.. i

 

 

LAB-6: Export isos to .dgn

 

1.       In the Management Console right click on Unit01 under ISOS and select Save As



2.       In the Save As form enter a valid Output Folder path

3.       Under File Types, for the Target File Type set iso drawings to MicroStation (dgn) and OK



4.       The 2D drawing editor will flash for a few seconds on the screen for each iso drawing that is processed. Dont close or interact with the 2D editor, allow for it to run until all drawings have been processed

5.       Go to the output folder to review the resulting .dgn files
