Workflow Review
•Create a folder in the Drawings and Reports task’s Management Console for new drawings
•Define Layout styles (by CAE)
•Define a Composed Drawing Component (holding place)
•Go to Space Management task to define volume(s) to associate with drawing view(s)
•Go to a 3D task such as Common and use the Tools > New Drawing (7.3) or Tools > Drawing Console (7.5+) command to create a new drawing
•Define ortho views in new drawing’s 2D Editor
•Link ortho views to 3D volumes
•Go to Drawings and Reports to update the drawing and perform additional editing
