Cable Routing
• In the Electrical task, you can create cables and then route those cables through existing conduit and cableway. The software provides the following features:
•Create cable and assign properties
•Ability to Create Parallel Cables
•Ability to automatically route cables on the shortest path from start point to end point. Uses set of rules to find the nearest cableways
•Editable cable routes including the ability to define where cables enter and exit cable tray
•Insert cable markers and generate a report of the cable route by marker name
•Cables are implied items that are not permanently displayed in 3D, but can be selected and seen as single line graphics
