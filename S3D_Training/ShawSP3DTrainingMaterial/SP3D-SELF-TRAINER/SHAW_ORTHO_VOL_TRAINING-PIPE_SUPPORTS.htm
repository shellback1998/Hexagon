PIPE SUPPORTS ORTHO EXTRACTION BY VOLUME

This procedure is time consuming, but useful in situations when graphic elements that are normally excluded from pipe support drawings by query need to be shown in a drawing. An example could be the graphics from an equipment item that the support must be attached to. By default supports drawings only show supports, piping, and structural elements. This procedure will capture any graphics from any other discipline that are within the range of the support.

In this example, the hanger’s drawing volume will be customized to show a complete pipe branch being supported.

  1. In SP3D, change tasks to Space Management
  2. Select the Create Space Folder command  and name it U01volumes. Click Finish to save
  3. Find the hanger or support to be extracted on a drawing, select it, and window in to it. This example will be using the one support in training plant’s unit U01
  4. Select the Place Volume by Selectset command (then click on the hanger if it wasn’t already selected). A wireframe box will appear around the hanger, it will turn solid once its attributes are defined and accepted.
  5. Give the volume a name, a good idea is the same name as the support, define Type as shown below and set space folder to U01volumes. Click Finish to accept.
  6. With the support’s volume box selected, revise the planes  of the volume to fully enclose the support, plus the top of the supporting beam, and the piping all the way to the tee.
    NOTE: If any graphic item that should not appear on the ortho falls inside the volume, a style rule can be applied that excludes this other type of item during drawing extraction.

  7. Next, create a drawing using this volume: Select Tools> Drawing Console
  8. Right click on Drawings, on the menu select New…
  9. On the Add Component form, select  Composed Drawings and OK
  10. Right click the New Composed Drawings component just created and select New Drawing
  11. Name it the same as the support, use the delivered layout and border templates as shown below, then OK
  12. A blank drawing template appears. Use the Place View command  to create 3 views on the drawing for elevations and iso views of the support. The steps below outline the view creation parameters.

  22. Select the Looking South view frame in the drawing
  23. Select Associate Objects to View command
  24. Bring to front the 3D environment and select the volume around the hanger/support
  25. Bring to front the ortho environment and select the save command . That links the view on the orthographic drawing with the 3D volume around the hanger.
  26. Repeat the view association process shown above for the East and Iso views
  27. Select the report view frame above the ortho iso view, then the Associate Objects to View command
  28. Bring to front the 3D environment and select the iso view, then Finish
  29. Bring to front the ortho environment and select the save command .
  30. Repeat the same procedure to associate the report view above the East elevation view with this same “east” view. This will produce a supporting structure report with a callout number in the East view.
  31. Repeat the same procedure to associate the report view above the South elevation view with this same“south” view. This will produce a piperun report with a callout number in the South view.
  32. Save and exit the ortho environment
  33. In 3D, select Tools>Drawing Console
  34. Expand the New Composed Drawings component created earlier, the new drawing appears below, right click on it and select Update Now


  35. When the update completes, and a green check mark appears on the drawing icon, right click the drawing and select Edit
  36. In the drawing, right click the South view’s frame, select Properties from the menu, then in the Format tab turn off the Show Border check box and OK
  37. Repeat for all the other graphic and report views to turn off their frames
  38. Drag reports to their proper location if needed
  39. Drag callout bubbles needing adjustment to their best location. New locations are preserved during future updates
