Creating a Custom Piping Spec
•
–In “CatalogData” workbook, locate piping parts needed, copy and modify as needed, most often commodity code and dimensions are adjusted per the new spec requirements
–Bulkload workbooks to catalog
–Run spec Verification and review report
–“Allow” spec for modeling and perform preliminary manual testing
–Repair, rebulkload and retest until all parts in the spec place as needed
