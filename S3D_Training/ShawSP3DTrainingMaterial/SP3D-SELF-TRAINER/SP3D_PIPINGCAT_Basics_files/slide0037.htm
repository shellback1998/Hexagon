Working With Code Listed Values
•A code listed value provides a standard attribute linked to a numeral that uniquely identifies that attribute
•Spec writer can add new code lists
•When modifying a code list, the spec writer must follow requirements for that list, e.g. some code lists require user-added values to be greater than 10,000; other code lists allow changing textual value but not numeric value or meaning
