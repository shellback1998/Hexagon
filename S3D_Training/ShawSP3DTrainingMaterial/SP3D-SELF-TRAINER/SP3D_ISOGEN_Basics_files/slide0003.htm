Overview
•SP3D uses Isogen by Alias, Ltd. to automate piping isometric drawings extraction •SP3D provides a set of customizable options or “drawing  styles” that control the drawing output, including format and content •Different style outputs can be extracted and maintained for the same pipelines, e.g., system isos, WBS isos, check isos, etc.
•All drawings are stored in the model database
•Iso drawings can be saved to multiple formats: sha, pcf, dxf, dwg, and dgn
