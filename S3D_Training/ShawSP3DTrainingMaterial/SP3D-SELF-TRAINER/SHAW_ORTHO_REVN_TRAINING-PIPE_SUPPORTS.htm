PIPE SUPPORTS ORTHO REVISION

VIEW REVISION

After extraction, the supports drawings may be annotated, and the views orientation can be revised. If such is the case, the Update command must be used for the orientation revision to take effect. Manual annotations will be preserved if placed in the Default layer.

  1. Edit the support ortho just extracted, change the Iso view orientation to Southwest.
    1. Open the drawing for edit
    2. Window-in to the iso view of the support
    3. Right-click the iso view’s frame and select Properties
    4. Change the view’s orientation to Looking Southeast and OK
    5. Save and exit the support drawing
    6. Right click the drawing just revised and select Update Now
    7. Once the update completes, enter the drawing and verify that the orientation was revised successfully
    8. Select View>Background Sheets from the main menu
    9. Edit the name of the view to “Isometric Looking South East” (double click the label to enter edit mode)
      WARNING: Running Update Now again on this drawing will reset this label back to “North East”, as this label comes from a template. Shaw templates will not have this orientation hard-coded, but Intergraph templates will.
    10. Select View>Working Sheets to return to the active drawing
    11. Save and exit the support drawing

 

TITLE BLOCK REVISION

To add title block information to the drawing, the information must be saved to the database first. Then the update-now command will take that information and add it to the drawings. This is advantageous in that the user will only enter a set of initials and dates once, for example, which in turn will automatically be propagated to all drawings in a hierarchy, whether a handful or hundreds of them.

  1. Populate the title block with initials, dates, and miscellaneous information.
    1. To modify title block properties, in the Management Console right click U01 and select Properties.  In there, set properties common to all drawings under U01.
    2. Under the Title Area tab set Company Name, Desc1, Desc2, Title1 and Title2 to any desired values.
    3. Under the Signature Area set Designed, Checked, and Approved fields to any desired values.  Information entered here will be inherited by all the drawings under U01at the next Update-Now job.


NOTE: In order to enter support properties specific to a single support drawing, the user will need to navigate down the full hierarchy, find the exact drawing needing the information changed, right click on it and use the Properties form.  Any data entered this way will override U01-wide or plant-wide generic information like that entered in the steps above.

    1. After using the Properties form, the document icon is switched from a green checked document to a crossed out document (meaning out-of-date). Run Update Now to incorporate the title block information into the drawing. After successful completion, enter the drawing in Edit mode and window in to the title block to review it.
