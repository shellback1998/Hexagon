Overview
•The Hangers and Supports Task Module is used to model supports/hangers for piping, ducts and trays
•A support is an assembly of parts, also called components, which connect together to the supported features, and to the supporting structure
•Can place two types of supports, Standard Supports and Designed Supports
•Hangers and supports are dependent on both the supporting object and the supported object
•Place the supporting and supported objects before placing any hangers or supports
•Tools are provided for automatic support drawing creation
