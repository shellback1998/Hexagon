SP3D Hangers and Supports Task
3
Overview
  • The Hangers and Supports Task Module is used to model supports/hangers for piping, ducts and trays
  • A support is an assembly of parts, also called components, which connect together to the supported features, and to the supporting structure
  • Can place two types of supports, Standard Supports and Designed Supports
  • Hangers and supports are dependent on both the supporting object and the supported object
  • Place the supporting and supported objects before placing any hangers or supports
  • Tools are provided for automatic support drawing creation
4
Hangers/Supports Task Commands
5
Place Support by Structure (Standard)
  • places a support at the intersection of the selected supported feature (or features) and a vertical plane passing through the selected supporting structure


  • Lab 1
6
Place Support by Point (Standard)
  • Places a support for a feature at a specific point along the feature
  • A support on a feature running under/above a slab or a plate can connect to the structure/slab at any point along the feature
  • The plane selected on the supporting structure can be at any angle


  • Lab 2
7
Place Support by Parts (Designed)
  • A designed support is a custom configuration of parts
  • First creates reference graphics at the identified logical support location instead of placing physical parts
  • Manually add support parts by using the Place Part command
  • Edit the position and the orientation of these parts using the Move and Rotate common commands
  • Labs 3-4
8
Drop Standard Support
  • Used to modify the individual parts of a standard support after placement
  • Users may convert a standard support to a designed support by using the Drop Standard command
  • This removes the control of the standard support over the parts so that they can be edited individually

Delete Support and Parts
  • Use correct locate tool filter to select a component or a support for deletion

    cause the steel member to be reported as

  • Labs 7-8
13
Editing Support Data Tips
  • Modify properties: Edit the reporting and other management properties of both the standard and designed supports. Can edit the dimensions of standard supports. Can edit the individual properties of parts within designed supports
  • Select different inputs: Can modify supports by selecting a different supported route feature, or a different supporting structure
  • Move: Can move a support positioned by point
  • Copy: Can copy a standard and designed support, and support parts to a different location
  • Delete: Can delete standard and designed supports. Cannot delete a component from a standard support because the parts are controlled by the standard support assembly. Use the Drop Standard Support command to break a standard assembly into editable parts
