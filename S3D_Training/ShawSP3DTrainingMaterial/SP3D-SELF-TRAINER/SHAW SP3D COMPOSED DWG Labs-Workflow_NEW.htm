1-Volume Workflow

A.   Creating the Component and Drawing


COMPONENT DEFINITION

1.      Enter the SP3D training plant.  In Drawings and Reports task, open/create the Unit 01/Orthos/Electrical drawings folder

2.      Right mouse click Electrical and select New

3.      In the Add Component dialog, from the General tab, select Composed Drawings and OK

4.      Rename the newly created drawing type to Training Composed Drawings

This drawing component can hold one or many ortho drawings as needed.

DRAWING DEFINITION

1.      Change task to Space Management task by selecting Tasks > Space Management from the menu bar

2.      Use File>Define Workspace  and define a workspace that displays the System part of the plant to be composed in a drawing.  In the following exercises we will extract a cable tray and a GA orthographic drawings.
For the first exercise set the workspace to Unit 1 of the Intergraph training model.
IMPORTANT: include the Named Space folders tab contents in your workspace definition filter (for this lab, select SP3DTrain in this tab).

3.      OK to define workspace.  Fit all graphics on a plan view and hide any volumes already existing (right click + Hide)

4.      Select the Create space folder option  and enter Name: UNIT01 with Parent: SP3DTrain and select Finish.  The Space tab in Workspace Explorer will display the new space folder

5.      While in the Create space folder mode  enter Name: Structural with Parent: SP3DTrain and select Finish.  This will be needed in a later lab.  The Space tab in Workspace Explorer will display the new space folder

6.      To create drawing views from a volume, use the Place Volume by Selection  command to define a drawing volume. 
As a minimum, define a volume with all the structural items in Unit 01 by selecting the slab on the bottom and one of the beams on the top of structure. Follow this procedure:

a.       Select Place Volume by Selection

b.      In ribbon bar enter Name: U1V1

c.       Under Type, select Definitions>Drawing Volumes>DWGVOLUME1

d.      Under Space folder set UNIT01

e.       Select Unit 01 slab in graphics

f.       Select a beam at the top of the structure

g.      Select Finish button to place the volume



The volume created will contain the graphics to be composed in the ortho drawing to be created

7.      OPTIONAL:  To adjust a side of the volume, i.e. the top did not cover some needed graphics for the drawing, do as follows:

a.       Click selection tool and set selection filter to Drawing Volume

b.      Orient your view to better visualize the volumes face to be adjusted, i.e. set view to Right:

c.       Click the volume and then click the Handles command

d.      Use QuickPick to select top edge

e.       Drag top edge to desired elevation and click position to accept (in this case, above cable tray)

f.       Volume size is adjusted

8.      While still in the Space Management task, select Tools > Drawing Console

9.      For the Location property of the new drawing, expand the list to select Orthos\Electrical\Training Composed Drawings created earlier. 

10.  Right click on Training Composed Drawings and select New Drawing

11.  For the Name property, type in Composed Drawings Lab1.  This will be the name of the physical drawing created in the component created above.

12.  For the Layout Template property, drop down the select list and choose More.  In the Select Template, select the Empty.sha template and click OK.

13.  For the Border Template property, drop down the select list and choose More.  In the Select Template dialog, navigate to the Shaw template for your discipline, if none available, double click the Imperial folder.  Select the D_Wide.sha template and click OK.

14.  Click OK on the Drawing Sheet General Properties dialog, which will open the Shape2D drawing editor behind SP3D.  Bring the Shape2D drawing editor up as the active application.

B. Placing the Views

 

1.      In Shape2d, fit the view and select the Place Region command

2.      Use single-clicks to place the region corners such that you get a box taking up about half of the drawing space inside the border

3.      In the Region Properties box enter name and description, for the Layout Style select Two View A and OK.  Your drawing should look something like this:

4.      Select Place View command  from the horizontal toolbar.

5.      With this command active, place a view anywhere inside the region frame using a two point method, roughly taking up half the regions area. 

6.      After the view is placed the Drawing View Properties dialog will display.  On the View tab of this dialog, drop down the Style select list and choose More.  This will open the Select View Style dialog.  In the Orthographic folder, select an Elevation style for your discipline, in this case Electrical Cable Tray Elevation and click OK.

7.      Notice, after the view style is selected, more options appear for the definition of this view.  It will look as follows:
 

8.      For the Name property, type in North1.

9.      For BOTH the coordinate systems, choose U01 CS or another suitable coordinate system to measure coordinate locations in this drawing

10.  For the Scale Family property, drop down the select list and choose Architectural Scales and set the ..Selected Scale to 1/8in=1ft

11.  Scroll down in this dialog and locate the property Look Direction.  This field should be set to Looking North

12.  In the Format tab un-check the Show Border box and select OK

13.  Once this dialog is closed the drawing template will automatically save the changes made to the template up to this point.  Notice that the view is saved and automatically fitted to the bottom half of the Region

14.  Place another view of random size within the upper half of the Region.

15.  Pick a Plan view style for this view according to the discipline, in our case Electrical Cable Tray Layout Plan

16.  Name this view Plan1, set the coordinates and scale information same as you did for the Elevation view.  Make sure the Look Direction is Looking Plan

17.  In the Format tab un-check the Show Border box and select OK

18.  Select Place View command  and place a third view on the right side of the drawing area completely outside the region.  This is called an Unmanaged view because it doesnt have an association to any regions on the drawing.

19.  In the Orthographic style folder select a View Style for Isometrics, i.e. Electrical Cable Tray Isometric

20.  Enter view property info and scale as before, make sure to set Look Direction to Looking NorthEast Down

21.  In the Format tab un-check the Show Border box and select OK

C. Associating the Volumes to Views

1.      After placing the views, the volume will need to be associated to them.  Choose the first view placed (View1), by clicking on the border of the view, and select the Associate Objects to View command  located on the horizontal toolbar.

2.      With this command active, switch back to SP3D, which should be open behind the Drawing Editor and notice the ribbon bar in SP3D. The ribbon bar should look as follows:
Notice that the View name and looking direction are the presets made from the view definition in the drawing editor (shape2d)

3.      Select the volume created earlier in this lab.  The volume will be highlighted in yellow.  
 With this volume selected, switch back to Shape2D and click the Save  button.  This has associated this 3D volume with the view selected in Shape2D.

4.      Choose the second view created earlier in Shape2D by clicking on the border of the view, and select the Associate Objects to View command  located on the horizontal toolbar

5.      For this view, select the same volume used earlier. This allows users to look at the same volume from multiple directions in a drawing

6.      With this volume selected, switch back to Shape2D and click the Save button.  This has associated this volume with the view selected in Shape2D.

7.      Repeat this procedure one more time to associate the iso (third) view with the same 3D volume and save your drawing in Shape2D.

8.      Close the Shape2D application.

D. Updating the Drawing

1.      Change to the Drawing Console by selecting Tools > Drawing Console from the menu bar of your current SP3D environment.

2.      Navigate to the new drawing just created in your disciplines orthos folder and select it.  In our earlier example for this exercise we created a component named Training Composed Drawings where we placed this training drawing.

3.      Notice the drawing created from this lab has the  icon next to it, meaning it hasnt yet been updated.  Right mouse click on the drawing and select Update Now.

4.      Once the drawing is Up-To-Date, the icon switches to .  Right click the drawing and select Edit to view/edit the results in Shape2D.

A. Creating a Volume by Object

 

1.      In the training plant, set the workspace (Ctrl+W) to show all plant items

2.      Go into the Space Management task by selecting Tasks > Space Management

3.      Select the Create Space Folder command

4.      Enter GA for the folder name

5.      The parent should be the plant root level, in this case SP3DTrain

9.      In the ribbon bar, name the volume ALLEQP

10.  Set the Type to: more>Definitions>Drawing Volumes>DWGVOLUME1 and OK

11.  Set the Space folder to: more>SP3DTrain>GA

12.  Select the outermost graphics that will define the size of the volume, for example, the highlighted items shown were selected to define this volume:

13.  Select Finish.  The volume becomes a solid. 

14.  Optionally, use Format>Style command to make the solid transparent

C. Labeling Equipment Items

1.      Select the Place a label command

2.      Select a graphic from any equipment item

3.      Select the pull down for labels and More

4.      Select label Equipment Plan Equipment_Name and OK

5.      Drag the label that appears to a desired location

6.      Click another equipment item and place its name label

7.      Select the dimension toolbar command

8.      The toolbar is activated, then select the Distance Between command

9.      Click on 2 different equipment items to place a measure between them.  In this example, the heat exchangers nozzles centers are selected to measure the distance between the items.

 

Note: SP3D Drawings and Reports provides Style Rules that can place item labels and dimensions like the ones above automatically.  Well take advantage of these features in the live projects, but from time to time it becomes more practical to use manual labels and dimensions.

E. (Optional) Applying Lessons to Live Project

 

1.      Switch to the live project

2.      Navigate to the Drawings and Reports task

3.      Go to an orthos folder to be designated by the administrator

4.      Define a drawing by query manager, review steps from previous labs for details

5.      In the Space Management task, create a volume over a select group of equipment items in order to create a GA drawing of their area

6.      Use Tools>Drawing Console to define a GA plan drawing that uses style Equipment Plan_M (M indicates metric units).  This time select a metric scale that may fit your drawing area

7.      Update and review the drawing contents.  Notice the automatic labels and automatic dimensions placed by the system.  You may need to move some labels, or delete others to clean out the drawing.  SP3D will remember any edits to labels or dimensions, so that they will not be moved at the next update, theyll remain where you relocate them to.
