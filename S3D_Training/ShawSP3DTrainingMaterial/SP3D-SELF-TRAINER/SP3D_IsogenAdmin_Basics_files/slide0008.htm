ISOGEN Admin Basic Concepts
•There are five iso styles delivered with the software that can be used for fast out of the box iso extraction, or as templates to modify and create company-specific styles:
–Pipeline
–Piperun
–Stress
–WBS
–Spool
•New styles can be created, but there is no option to create isos by other attributes than these
•Intergraph recommends using WBS for production isos. However, the procedure is complicated by an intermediate step of creating WBS packages
•The administrator can setup iso extractions by pipeline, piperun, or spool for checking purposes separate from the isos to be IFC
•The stress style produces a Piping Component File (PCF) that can be imported directly by CAESAR II for stress analysis
