Terms to Know
•These terms describe the different tray and conduit components that you work with during a session with the software: • A feature is a high-level component that composes part of a run in the model. An example is a tee. You are free to move the feature at any time during the design process • A turn feature is a high-level component that composes part of a run in the model. The turn feature represents a change in direction for the cableway or conduit run • An end feature is a high-level component that represents either end of the cableway or conduit run • A straight feature is a high- level component that represents a straight section of cableway or conduit • A transition feature is a high- level component that represents a change of cross section of the cableway • A parent feature is a high-level component that represents a feature with an additional feature. An example is an insulated straight feature
